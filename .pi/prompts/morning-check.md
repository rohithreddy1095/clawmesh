Perform a comprehensive morning status check for the farm:

1. **Sensor scan:** Query all recent observations from the world model (last 50 frames).
   Summarize current conditions for each zone — moisture, temperature, any anomalies.

2. **Connectivity:** List all connected mesh nodes and their capabilities.
   Flag any nodes that should be online but aren't reporting.

3. **Proposals:** Check all pending proposals. Summarize any awaiting approval.

4. **Health summary:** Provide a concise farm health assessment:
   - Any zones in critical condition?
   - Any sensors reporting unusual values?
   - Any infrastructure issues?
   - Recommended actions for today

Be concise. Use tables where helpful. Flag anything urgent.
