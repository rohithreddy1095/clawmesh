Mesh Protocol

discovery · trust · routing

peer connectivity

Discovery & Trust

  • _clawmesh._tcp mDNS service scan
  • Ed25519 signed handshake payloads
  • File-backed trust store with atomic writes
  • Discovered peers ignored until explicitly trusted

capability routing

Local-First

  • Dynamic capability registry
  • channel:* skill:* actuator:*
  • Route by advertised caps, not plugins
  • RPC-based message forwarding

Context Engine

emergent intelligence via gossip

propagation

No File Sync. Just Gossip.

  • Sensor observations become ContextFrames
  • Frames broadcast via WebSocket gossip
  • Each node maintains a WorldModel
  • Planner LLM reasons over mesh-wide context
  • No manual coordination needed

05

Intelligence

pi-powered planning + pattern learning

planner

Pi-Powered LLM

  • Multi-provider AgentSession
  • Reasons over mesh WorldModel
  • Creates TaskProposals with levels
  • Forwards execution to capable peers
  • Farm domain knowledge injection

learning

Pattern Memory

  • Captures approve/reject decisions
  • Correlates to trigger conditions
  • Patterns mature over repetition
  • Mature patterns gossip to mesh
  • Persisted to ~/.clawmesh/
